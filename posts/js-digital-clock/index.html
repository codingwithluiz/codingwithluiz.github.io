â²ï¸ Step 3: Add the JavaScript Logic

Create a file named script.js:

function updateClock() {
  const now = new Date();
  const hours = String(now.getHours()).padStart(2, '0');
  const minutes = String(now.getMinutes()).padStart(2, '0');
  const seconds = String(now.getSeconds()).padStart(2, '0');

  const timeString = `${hours}:${minutes}:${seconds}`;
  document.getElementById('clock').textContent = timeString;
}

// Initial call to show clock immediately
updateClock();

// Update every second
setInterval(updateClock, 1000);

Breakdown:

  • new Date() gets the current system time.
  • padStart(2, '0') ensures 2-digit formatting (e.g., 08 instead of 8).
  • setInterval updates the clock every 1000 milliseconds (1 second).
